Features
========
* Updates Factorio Mods from FactorioMods.com
* Installs .factoriomod files
* Installs Factorio Mods through the factoriomods uri protocol

Installation
============

Releases coming soon. Program is a work in progress.

============

* Python 3.4
* appdirs
* requests
